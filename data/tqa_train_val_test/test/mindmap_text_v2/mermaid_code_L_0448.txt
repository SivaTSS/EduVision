        Single-celled eukaryotes

        Traits shared with animals

      Five Kingdom System
        Proposed by Whittaker
      Kingdom Protista

    Groups of Protozoa
      Flagellates
        Long flagella
        Example: Trypanosoma
      Ciliates
        Move using cilia
        Example: Paramecium
      Sporozoans
        Produce spores
        Example: Toxoplasma
      Pseudopodia
        Temporary feet-like structures
        Example: Amoeba
